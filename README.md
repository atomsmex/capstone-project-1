# Static Website Deployment on AWS
This project demonstrates a professional hosting setup using **Amazon S3** for storage and **Amazon CloudFront** for global content delivery.

## Project Features
* **S3 Hosting:** Configured for static website hosting with public read access.
* **CloudFront CDN:** Used to distribute the site globally with lower latency.
* **Security:** Implemented bucket policies for controlled access.
